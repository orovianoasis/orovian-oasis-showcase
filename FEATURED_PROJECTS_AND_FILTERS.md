UPLOAD ONE EXTRACTED PROPERTY FOLDER INTO content/projects/

The folder should contain:
- cover.webp
- README.txt
- walkthrough HTML
- GLB model
- DXF floor plans

The original long filenames and underscores are allowed.
project.toml is optional.
Do not upload the ZIP itself.


PROJECT COVER SLIDESHOWS (v3.4)
Add any of these directly inside the one uploaded property folder:
- cover-front.webp
- cover-back.webp
- cover-01.webp, cover-02.webp, and so on

Every filename beginning with "cover" becomes part of that project's carousel.
With no cover image, the generated placeholder remains.
Global autoplay is controlled in content/site.toml.
