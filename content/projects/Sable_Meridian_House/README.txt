SABLE MERIDIAN HOUSE — COMPLETE EDITABLE PROPERTY PACKAGE

PROJECT
Sable Meridian House is an original two-story desert-modern courtyard estate for a high-desert climate. Three offset bars protect a central sun court while framing north-facing pool and mountain views. The package is dimensionally coordinated across DXF, OBJ, GLB, and the embedded HTML walkthrough.

DESIGN SUMMARY
- Approx. 9,180 square feet conditioned
- Approx. 12,860 square feet under roof
- 5 bedroom suites
- 6 full bathrooms and 1 half bathroom
- 3-car garage
- South arrival, north pool court, east driveway
- Double-height arrival gallery, sculptural stair, elevator, wellness wing, library lounge, media room, creative studio, shaded outdoor kitchen, linear pool, spa, and fire terrace
- Climate response: deep shade, protected courtyard, flat parapet roofs, low-e glazing zones, desert landscaping, and screened western exposure

COORDINATE / UNIT CONTRACT
- Common origin: southwest corner of the west wing at DXF coordinate 0,0
- Orientation: +X east, +Y north, +Z up
- DXF files: inches
- OBJ, GLB, HTML geometry, walk surfaces, and collision boxes: meters
- Exact conversion: 1 foot = 0.3048 meter

DXF CONTENT
The DXFs use organized architectural layers for exterior walls, interior walls, doors, windows, fixtures, furniture, casework, stairs, room labels, dimensions, site, pool, notes, and drawing-index information. They are editable ASCII DXF files with inch units and common registration.

3D CONTENT
The OBJ is a broad editable geometry export with meter units, face normals, vertex color values, and separate main / upper / roof groups. The GLB is a compact glTF 2.0 scene with coordinated vertex colors and the same origin and scale. Modeled content includes architecture, slabs, roofs, stairs, elevator, cabinetry, plumbing fixtures, lighting, appliances, furniture, vehicles, hardscape, pool/spa, terraces, landscape elements, and site features.

WALKTHROUGH
Open Sable_Meridian_House_Interactive_Walkthrough.html directly in a current Chromium-based browser after extraction. It contains embedded vertices, normals, colors, main indices, roof indices, upper-slab indices, collision boxes, floor zones, stair ramp data, renderer code, and controls. It makes no network requests and does not load the GLB.

DESKTOP CONTROLS
Arrow keys / WASD = move
Mouse = look
Q + arrow keys = keyboard look / turn
Shift = run
F = Walk / Fly toggle
E = exterior aerial view and Fly mode
Space / C = rise / lower in Fly mode
R = roof visibility
U = upper-slab visibility
X = reset to arrival and Walk mode
I = info panel
T = tools panel
Mouse wheel = zoom

MOBILE CONTROLS
Centered D-pad = move
Drag scene = look
Pinch = zoom
Bottom-left: EXTERIOR / ROOF above SLABS / RESET
Bottom-right: RUN / UP above FLY / DOWN
UP and DOWN are disabled in Walk mode and enabled in Fly mode. FLY, ROOF, SLABS, INFO, and TOOLS retain visible state.
