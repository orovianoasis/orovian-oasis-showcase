DESKTOP WALKTHROUGH CONTROLS
Open the HTML file in a current desktop browser and select Enter Walkthrough.
Arrow Up / W = forward
Arrow Down / S = backward
Arrow Left / A = strafe left
Arrow Right / D = strafe right
Mouse = look
Shift = run
F = fly/walk
Space / C = rise/lower in fly mode
R = roof visibility
U = upper-slab visibility
X = reset

MOBILE WALKTHROUGH CONTROLS
Open the HTML file in a current mobile browser.
On-screen arrow pad = move
Drag the right side of the scene = look around
RUN = faster movement
FLY = toggle fly/walk
RISE / LOWER = vertical movement in fly mode
ROOF = roof visibility
SLABS = upper-slab visibility
RESET = return to the starting position

The walkthrough is self-contained inside the HTML file. Some phone file-preview apps block JavaScript or WebGL; in that case, use the phone's Share/Open In command and open the file directly in Safari, Chrome, or another current browser.

MODEL CONCEPT
Original two-story organic-modern luxury estate with approximately 5,850 square feet of conditioned living area, four bedroom suites, 4.5 bathrooms, three-car garage, double-height great room, glass courtyard, library bridge, private gym, media lounge, resort pool, spa, covered outdoor kitchen, firepit lounge, and furnished interiors.

UNITS
DXF files: inches. 3D model: meters, using 1 foot = 0.3048 meter.

IMPORTANT
This is a detailed architectural concept model and drawing set, not a stamped construction document. Structural engineering, mechanical/electrical/plumbing design, local code analysis, site-specific grading, energy compliance, and permit drawings must be completed by licensed professionals before construction.
