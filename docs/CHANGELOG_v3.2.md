# Version 3.2 — One-folder project importer

- Rebuilt from the unmodified-project v2.9 source package.
- Added automatic raw property detection under `content/projects/`.
- Made `project.toml` optional.
- Added automatic README metadata parsing.
- Added automatic hyphenated public slugs from underscored or spaced folder names.
- Added walkthrough HTML and GLB detection without source-file renaming.
- Added generated tour `viewer.html`, `house.glb`, `collision.json`, and `manifest.json` in the deployment output.
- Added dependency-preserving walkthrough file copying.
- Added automatic DXF-to-SVG browser previews and original DXF downloads.
- Added automatic Structure Demo suppression after a real project is uploaded.
- Preserved the existing advanced `project.toml` workflow.
