IMPORTANT TEST BOUNDARY
No Chief Architect DAE export was included in the supplied repository, and a full Blender installation was unavailable in the artifact environment. The GitHub workflow and conversion command were therefore tested with a controlled Blender-command harness rather than a real Chief-exported DAE. The workflow installs Blender 3.x on Ubuntu 22.04 and executes scripts/dae_to_glb_blender.py when a DAE is committed.

Version 3.8 one-folder contract:
- Export a Collada/DAE model from Chief Architect and keep its textures beside it or in a textures folder.
- Upload one extracted property folder into content/projects/.
- Add property-details.json for one-file control of public data and optional walkthrough tuning.
- GitHub Actions converts the DAE to GLB and connects the permanent custom walkthrough automatically.
- Existing HTML/GLB tours remain supported unchanged unless walkthrough.prefer_dae is true.
- Generated walkthrough files are build products and do not need to be uploaded or committed.
