# Release checklist

- [ ] Folder name and `slug` match
- [ ] Correct category selected
- [ ] Price and availability are current
- [ ] Checkout and inquiry links work
- [ ] Public title, summary, facts, and location are reviewed
- [ ] Cover, card, and sharing media are current
- [ ] Floor-plan images are customer-facing and legible
- [ ] DXF files are only public when intentionally offered
- [ ] Materials and deliverables do not overpromise
- [ ] Tour assets are complete before `tour.enabled = true`
- [ ] No private client or acquisition data is present
- [ ] `published = true` only when ready
- [ ] Pull-request validation or local build passes
