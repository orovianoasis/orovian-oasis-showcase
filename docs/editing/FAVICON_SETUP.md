# Favicon Setup

The browser-tab icon is controlled by:

```text
assets/brand/favicon.png
```

Replace that file with your own **square PNG**. Recommended size: **512 × 512 pixels**. Keep the exact filename `favicon.png`; the site layout already links to it, so no HTML editing is required.

The same file is also used as the Apple touch icon. After replacing it, rebuild and redeploy the site. Browsers sometimes cache favicons aggressively, so a hard refresh or private window may be needed before the new icon appears.
