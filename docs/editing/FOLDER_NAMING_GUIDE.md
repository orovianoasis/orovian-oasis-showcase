# Folder naming guide

- `website/` contains all editable page HTML, CSS, and JavaScript.
- `assets/` contains global visual media such as PNG, JPG, WebP, SVG, logos, icons, and thumbnails.
- `content/` contains site settings and frequently updated project data.
- `content/projects/SLUG/assets/` contains media belonging to one project.
- `content/projects/SLUG/floor-plans/` contains customer-facing plan images and downloads.
- `content/projects/SLUG/tour/` contains the four-file browser walkthrough package.
- `scripts/` contains build and validation automation.
- `tools/` contains convenient Windows launchers.
- `dist/` is generated output and must not be edited manually.
